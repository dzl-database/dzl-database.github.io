使い方ガイド

非公式ファンサイト「ドズル社マップplus」の基本的な使い方を紹介します！
初めての方はぜひご覧ください！

map マップの見方

  • マップ上のピンをタップすると詳細情報が表示されます
  • ピンの色やアイコンでカテゴリが分かります
  • ピンをタップ → 「マップ」でその場所へズームできます
  • その他にもGoogle マップを使用した経路や公式サイトのリンクボタンもあります

search 検索機能

  • 右上の「🔍」ボタンで検索ができます
  • スポット名 / 住所 / イベント名 で検索可能
  • 現在地から近い順で表示されます

tune フィルター機能

  • カテゴリごとに表示・非表示を切り替え可能
  • 開催状況（開催前・開催中・終了）でも絞り込みできます
  • 色がついている状態がONです

list 近くのスポット一覧

  • 「リスト」ボタンで周辺スポットを表示
  • 現在見ている位置から近い順で並びます

my_location 現在地表示

  • 現在地ボタンで自分の位置を表示できます
  • 青い丸が現在地です

皆様に安心してお使いいただくため、以下の行為はご遠慮ください。

■ 情報提供（投稿データ）の取り扱いについて

  • 皆様からお寄せいただいた地点情報やコメントなどの著作権は、投稿されたご本人様に帰属します。

■ 免責事項
